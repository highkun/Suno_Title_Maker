Suno Title Maker

ワードを自由に入力するだけで、プロンプト・歌詞・ジャンル・ムード・シーン・楽器などから曲タイトルを生成できます。

Gemini APIキー

モデルは gemini-2.5-flash 固定です。APIキー取得はこちら（このツールはブラウザ上で動作します / APIキーはブラウザに保存されます）

入力

プロンプト、歌詞、ジャンル、ムード、シーン、楽器などを自由に入力できます。
ヒント：歌詞や雰囲気ワードを組み合わせると精度が上がります

生成

※ サンプルAはライト入力例 / サンプルBはSuno形式＋Lyricsの記入例です
入力して Generate を押すとタイトルが生成されます
高品質モード
→ タイトルを2回生成して精度を上げます

売れ線モード
→ Spotify / YouTube向けに寄せて調整します

量産モード
→ タイトル数を増やしてまとめて出します

タイトル長さ
→ 短い / 標準 / 長い を選べます

AIっぽさ排除
→ 自然で人間的なタイトルを優先します

言語
→ 日本語 / 英語 / 日英ミックスで出力を切り替えます
